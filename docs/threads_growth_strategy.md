## 1. Threadsアルゴリズムの仕組み

### 核心: 「会話」中心の設計

Metaが公式に明言している通り、Threadsのアルゴリズムは**「会話を生み出す投稿」**を最も高く評価する。
Threads全体のビューのうち**50%以上が返信コンテンツ**で占められており、一方的な情報発信ではなく対話型のコンテンツが圧倒的に有利。

### 4つの評価ステージ（Meta公式発表）

```
ステージ1: 初期露出（投稿後15分間）
  → 少数のユーザーに表示し、反応を測定

ステージ2: 拡散判定（投稿後15分〜1時間）
  → 初期反応が閾値を超えると「バズ候補」としてタグ付け
  → 反応者のネットワークへ露出を拡大

ステージ3: 本格拡散（1時間〜24時間）
  → エンゲージメント率が高ければ「おすすめ」フィードに掲載
  → 非フォロワーにも大量露出

ステージ4: ロングテール（24時間〜数日）
  → 会話が継続している投稿は長期間表示され続ける
```

### アルゴリズムが評価する3つの鍵

| 評価要素 | 重要度 | 詳細 |
|---------|--------|------|
| **読まれている時間** | 最重要 | スクロールを止めて長く読まれる投稿が高評価 |
| **コメントの質** | 重要 | 単なる「いいね」よりコメント、さらに会話の深さが重要 |
| **投稿の一貫性** | 重要 | 特定テーマに絞った継続投稿がアカウント評価を上げる |

### 2025-2026年の重要な変更点

- **品質重視シフト**: スパム的な大量投稿より「質の高い投稿」を優先（Meta公式方針）
- **エンゲージメント・ベイト対策**: 「いいねしたら○○」等の釣り投稿はペナルティ対象
- **フォロー優先**: おすすめされる投稿は、フォローしている人の投稿が優先される仕様に変更

## 2. バズる投稿の特徴

### 文字数

| 文字数 | エンゲージメント率 | 推奨用途 |
|--------|------------------|---------|
| **100〜200文字** | 最も高い | 日常投稿、レビュー要約 |
| 200〜300文字 | 高い | ストーリー型、比較投稿 |
| 300〜500文字（上限） | 中程度 | 詳細レビュー、ノウハウ系 |

### 画像・メディアの効果

- **画像付き投稿**: テキストのみと比較して**約40%高いエンゲージメント**
- 画像は最大10枚、動画は最大5分まで対応
- 商品写真は「使用シーン」が最も反応が良い

### バズる投稿の構成パターン

**パターンA: 結論先行型（最推奨）**
```
[衝撃的な結論/発見] ← 最初の5秒で目を引く
↓
[具体的な理由/体験] ← 信頼性を担保
↓
[問いかけ] ← エンゲージメント誘発
```

**パターンB: 比較型**
```
「AとB、どっちがいい？」
↓
[それぞれのメリット・デメリット]
↓
「あなたならどっち選ぶ？」
```

**パターンC: 発見共有型**
```
「○○使ってみたら衝撃だった」
↓
[Before/After、具体的な変化]
↓
「他に試した人いる？」
```

### 具体例（美容家電ジャンル）

```
【パターンA例】
「3万円のドライヤー、正直バカにしてた。

でも1ヶ月使ったら美容院帰りの髪が
毎日続くようになった。

速乾だけじゃなくて、髪質が変わる。
これが技術の進化か...

みんな高めのドライヤー使ってる？
おすすめあったら教えて！」(147文字)
```

## 3. フォロワー最速増加の具体的戦略

### なぜ今Threadsなのか

- 月間アクティブユーザー数**4億人超**（2025年時点）
- X(Twitter)と比較して**競合が少なく伸びやすい**
- フォロワー0からでも「おすすめ」に載る仕組みがある
- **1〜2週間で数千フォロワー達成**の事例が多数報告されている

### 戦略1: 初動15分を制する

投稿後15分間の反応数がバズの閾値を決める。

**実行方法**:
- 投稿直後はアプリを開いたままにする
- 来たコメントには即座に返信する（会話を生む）
- 投稿前に他のアカウントにリプライしておき、自分のタイムラインに人を集めておく

### 戦略2: リプライ営業（最重要戦略）

フォロワー0の段階で最も効果的なのは**他アカウントへの積極的なリプライ**。

**実行方法**:
- 同ジャンル（美容、ガジェット）のインフルエンサーの投稿にリプライ
- 「すごい！」ではなく「自分の体験+質問」を含む価値あるリプライ
- 1日最低20〜30件のリプライを目標にする
- リプライ先の投稿が新しいほど効果的（投稿後30分以内が理想）

### 戦略3: 共感のスレッド構築

**3ステップルーチン**:
1. **共感投稿**: 感情を揺さぶる体験談で反応を得る
2. **交流**: コメントへの返信で関係性を深める
3. **毎日投稿**: 露出を切らさずアルゴリズム評価を維持

### 戦略4: ニッチ特化で権威性を構築

「美容家電・ガジェット」の中でも更に絞る。

**推奨サブニッチ**:
- 「1万円以下の神コスパ美容家電」
- 「男が使って良かった美容ガジェット」
- 「Amazonで買える美容家電の闇と光」
- 「美容家電ガチレビュー（忖度なし）」

### 戦略5: コンテンツのシリーズ化

定期的なシリーズを持つことでフォローの理由を作る。

**シリーズ案**:
- 「今週の美容家電ベスト3」（週1回）
- 「1ヶ月使い込みレビュー」（月1回）
- 「○○ vs ○○ ガチ比較」（週1回）
- 「フォロワーの質問に答える」（週1回）

## 4. 最適な投稿頻度と時間帯

### 投稿頻度

| 段階 | 推奨頻度 | 理由 |
|------|---------|------|
| **初期（0〜1000人）** | 1日2〜3回 | 露出を最大化、アルゴリズムに認知させる |
| **成長期（1000〜5000人）** | 1日1〜2回 | 質を維持しつつ継続 |
| **安定期（5000人〜）** | 1日1回+リプライ活発 | 質重視、コミュニティ維持 |

Meta公式推奨は**週2〜5回**だが、初期フェーズでは毎日投稿が有利。
ただし**質を落としてまで量を増やすのは逆効果**（2026年のアルゴリズム変更により）。

### 最適な投稿時間帯（日本市場）

| 時間帯 | 効果 | 理由 |
|--------|------|------|
| **7:00〜8:00（朝）** | 最高 | 通勤・通学時間、エンゲージメント率+30% |
| **12:00〜13:00（昼）** | 高い | 昼休み、スマホチェックのピーク |
| **20:00〜22:00（夜）** | 高い | 初速が最も出やすい時間帯 |
| 土日 10:00〜14:00 | 中〜高 | 週末のリラックスタイム |

### 推奨投稿スケジュール（初期フェーズ）

```
平日:
  7:00  → メイン投稿（レビュー、比較、ノウハウ）
  12:30 → 軽い投稿（日常、つぶやき、質問系）
  21:00 → エンゲージメント重視投稿（議論、アンケート型）

土日:
  10:00 → 詳細レビュー投稿
  15:00 → カジュアル投稿
```

**重要**: 投稿時間よりも投稿の質が最終的なリーチを決める。時間帯は「初速」を有利にする補助要素。

## 5. Instagram連携による相乗効果

### 基本連携のメリット

- Instagramのフォロワーが自動的にThreadsのフォロー候補として表示される
- InstagramプロフィールにThreadsへの導線が表示される
- Instagramのフィードに人気のThreads投稿が表示されることがある

### 相乗効果を最大化する戦略

**戦略A: Threads → Instagram 誘導**
```
Threads: 短い感想・レビュー要約（テキスト中心）
  ↓ 「詳細レビューはインスタで！」
Instagram: 写真/動画による詳細レビュー
  ↓ ストーリーズでアフィリエイトリンク
収益化
```

**戦略B: Instagram → Threads 誘導**
```
Instagram: ビジュアル重視の投稿
  ↓ 「本音トークはThreadsで！」
Threads: 忖度なしレビュー、裏話、本音
  ↓ フォロワーとの深い関係構築
```

**戦略C: クロスプラットフォーム連携**
```
同じ商品を両方で紹介:
  Instagram → ビジュアル（写真/Reels）
  Threads → テキスト（使用感、比較、議論）
  → リーチ数が通常の1.5倍に伸長した事例あり
```

### 具体的な運用フロー

1. **Instagram投稿をThreadsにシェア**: Meta側がリーチを優遇する傾向あり
2. **Threadsで会話 → Instagramの投稿に誘導**: 「写真で見たい人はインスタへ」
3. **Instagramストーリーズ**: 「Threadsで○○について語ってます」とリンク
4. **プロフィール最適化**: 両方のプロフィールに相互リンクを設置

## 7. エンゲージメント施策

### リプライ戦略（フォロワー0からの最重要施策）

**Phase 1: 種まき（1〜7日目）**
- 美容家電・ガジェット系アカウントを50個以上フォロー
- 1日30件以上の「価値あるリプライ」を投稿
- リプライは「共感+自分の体験+質問」の3点セット

**Phase 2: 認知拡大（8〜14日目）**
- リプライ先のアカウントからフォローバックを獲得
- 自分の投稿への初期リアクションが増え始める
- 引き続き1日20件以上のリプライを継続

**Phase 3: 自走開始（15日目〜）**
- 自分の投稿がおすすめフィードに載り始める
- リプライ量を1日10件程度に減らし、投稿の質に注力
- コミュニティが自然に形成される

### リプライの質を高めるコツ

**良いリプライの例**:
```
相手の投稿: 「新しいドライヤー買った！めっちゃいい」
良いリプライ: 「どのモデルですか？自分も○○使ってるんですけど、
               温度調整の細かさが気に入ってます。
               比較してどうですか？」
```

**悪いリプライの例**:
```
「いいですね！」「最高！」「フォローしました！」
→ 会話が生まれない＝アルゴリズム的に無価値
```

### エンゲージメントを高める投稿テクニック

1. **質問で締める**: 投稿の最後は必ず問いかけで終わる
2. **選択肢を提示**: 「AとB、どっち？」形式は回答率が高い
3. **議論を誘発**: 「○○って本当に必要？」等の問題提起
4. **体験の共有を求める**: 「みんなのおすすめ教えて」
5. **投稿後1時間は張り付く**: コメントへの即レスが会話を拡大させる

## 8. やってはいけないNG行動

### アルゴリズムペナルティ対象

| NG行動 | リスク | 代替策 |
|--------|--------|--------|
| **エンゲージメント・ベイト** | 露出制限 | 自然な質問・議論で反応を得る |
| 「いいねしたら○○」系の釣り投稿 | ペナルティ | 価値提供型のCTAに変更 |
| 大量のハッシュタグ（6個以上） | スパム判定 | 2〜3個の関連性の高いタグに絞る |
| 短時間での大量投稿 | スパム判定 | 最低2時間間隔を空ける |
| コピペ投稿の繰り返し | 露出制限 | 毎回オリジナルのコンテンツを作成 |

### 成長を阻害する行動

| NG行動 | 問題点 | 改善策 |
|--------|--------|--------|
| フォロー/アンフォロー戦法 | アカウント信頼度低下 | リプライ戦略で自然増加を狙う |
| 投稿して放置 | 会話が生まれない | 投稿後1時間は返信に専念 |
| テーマの一貫性がない | フォロー理由がない | ニッチ特化を徹底 |
| 宣伝ばかりの投稿 | フォロー解除される | 宣伝:価値提供 = 2:8の比率 |
| 他人の投稿のパクリ | 信頼喪失、報告リスク | 自分の体験ベースで発信 |

### Meta共通の厳格化傾向（2025-2026年）

- 軽微な利用規約違反でもシャドウバンの対象になるケースが増加
- 著作権侵害（他人の画像無断使用）は即ペナルティ
- スパム報告が複数入るとアカウント制限の可能性
- AI生成コンテンツの過度な使用に対する監視強化の兆候

## 9. Threads広告・プロモーション機能の現状

### 広告機能のタイムライン

| 時期 | 状況 |
|------|------|
| 2025年1月 | 米国・日本で一部ビジネス向けテスト開始 |
| 2025年4月 | 全世界の広告主が利用可能に拡大 |
| 2026年3月（現在） | インフィード画像・動画広告が利用可能 |

### 現在の広告仕様

- **形式**: 画像広告、動画広告（インフィード表示）
- **設定**: Meta広告マネージャから既存キャンペーンを拡張する形
- **表示**: 通常投稿の間に「広告（Sponsored）」として表示
- **ターゲティング**: Metaのファミリーアプリ（Facebook/Instagram）のデータを活用

### アフィリエイターとしての活用

**現時点の推奨**:
- 個人アフィリエイターが広告出稿するメリットはまだ限定的
- まずはオーガニック（自然投稿）でフォロワーを増やすことに集中
- フォロワー1000人を超えたら、特定の高単価商品に対して少額テスト広告を検討
- Instagram広告との併用でコスト効率を最大化

## 信頼度と注意事項

**情報の信頼度**: 高（Meta公式発表、大手SNSマーケティングメディア、実績ある運用者の事例に基づく）

**注意事項**:
- フォロワー増加のスピードはジャンル・投稿の質・時期により大きく変動する
- 上記のKPIは好条件での目安であり、保証値ではない
- Threadsのアルゴリズムは頻繁に変更されるため、定期的な情報更新が必要
- フォロワー購入等のグレーな手法は長期的に逆効果になるため推奨しない
